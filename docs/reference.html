Add a data cell to the Display Frame Row.

Parameters:

  • align (str, optional) –

    ”left” : left-align the data, “right” : right-align the data, “center” : center-align the data, “fill” : fill the complete cell width with data repeated.
  • colSpan (int, optional) – Number of columns the cell spans for.

DisplayFrameException – If align is not a valid keyword or if colSpan is not a positive Integer.
